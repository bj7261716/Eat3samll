# Assets 資料夾

此資料夾用於存放應用程式的靜態資源檔案。

## 目錄結構

- `icons/` - 應用程式圖示和料理類型圖示

## 說明

目前應用程式使用 Emoji 作為圖示，因此不需要額外的圖片檔案。
如果未來需要自訂圖示，可以將 SVG 或 PNG 檔案放置在此資料夾中。

## 建議的圖示資源

- [Flaticon](https://www.flaticon.com/) - 免費圖示資源
- [Font Awesome](https://fontawesome.com/) - 圖標字體
- [Material Icons](https://fonts.google.com/icons) - Google 材質設計圖示
